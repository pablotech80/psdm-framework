Guide

From zero to a governed boundary.

01

Install Riscala

npm install -g @ptechsolution/psdm-framework@beta
02

Define a verifiable objective

riscala work init "Add email validation" --mode implement

Avoid generic placeholders. Describe the expected outcome.

03

Connect Codex

riscala adapters init codex
04

Change chats without rebuilding context

Open Codex in the same repository and say “continue.” It reads .riscala/ACTIVE_WORK.md, evaluates the boundary, and restores the handoff.

Mental model

Five pieces. One source of truth.

Repository
The authorized location before following any requested path.
Objective
The concrete outcome of the work.
Mode
Inspect, design, implement, or release.
Scope
Allowed and forbidden paths and actions.
Handoff
Progress, decisions, questions, and the next action.
