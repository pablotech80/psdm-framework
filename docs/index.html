Guía

De cero a un límite gobernado.

01

Instala Riscala

npm install -g @ptechsolution/psdm-framework@beta
02

Define un objetivo verificable

riscala work init "Añadir validación del correo" --mode implement

No uses marcadores genéricos. Describe el resultado esperado.

03

Conecta Codex

riscala adapters init codex
04

Cambia de chat sin reconstruir el contexto

Abre Codex en el mismo repositorio y di «continúa». Leerá .riscala/ACTIVE_WORK.md, evaluará el límite y recuperará el traspaso.

Modelo mental

Cinco piezas. Una fuente de verdad.

Repositorio
El lugar autorizado antes de seguir cualquier ruta.
Objetivo
El resultado concreto del trabajo.
Modo
Inspeccionar, diseñar, implementar o publicar.
Alcance
Rutas y acciones permitidas y prohibidas.
Traspaso
Progreso, decisiones, preguntas y siguiente acción.
